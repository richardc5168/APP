練習模式

（按「出新題」開始）
提示：答案格式支援「逗號或空白分隔」，可輸入分數如 1/2。
這頁怎麼做到「離線驗證」？
這個頁面在產題時「先選根 → 展開成整係數二次式」，因此可用判別式與公式解在前端直接驗證； 題庫驗證面板亦限制標準型：ax^2+bx+c=0（整數係數）。

題庫驗證（Quadratic Only）

支援標準型：ax^2+bx+c=0（可有空白；左右也可移項但 RHS 僅限常數）。答案格式："2, 3" 或 "1/2, -3"。

若要 Web 連結（GitHub Pages）
把這個檔案放在 GitHub repo 的 docs/quadratic/index.html， 然後到 repo 的 Settings → Pages → Build and deployment： 選 Deploy from a branch，Branch 選 main，Folder 選 /docs。 等幾分鐘就會有類似：https://<user>.github.io/<repo>/quadratic/ 的連結。
